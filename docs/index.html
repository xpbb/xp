欢迎使用 xpjs

xpjs是一个javascript的组件库，xpjs的目标群体是前端设计师或前端工程师，设计目标是：让用户不用书写任何js的前提下完成一系列复杂的web操作， 尽量做到轻巧、简单、易用、稳定、兼容。使用xpjs可以减少代码的编写量，并将你的精力投入到项目的创造性设计上。

xpjs类库特点：

  • 易用性：对于设计师来说，这是一套您不需要写任何js的js框架。xpjs尽量做到让用户少书写代码，多注重设计。 用户只要书写简单的html不用写js即可完成复杂的web操作逻辑。
  • 模块化：对于前端开发人员来说，xpjs的类管理完成了类继承、依赖、接口、外部自动加载等一系列的类对象功能，适合于大型js项目的开发。
  • 可扩展性和调试性：xpjs具有完善的可扩展机制和错误排查机制，对二次开发人员来讲，尽量做到有错能查，查到能改。
  • 最小粒度开发：能分解的尽量分解，借助类管理机制可以让复杂的逻辑轻松解耦。
  • dom和原生函数零污染：dom和原生函数的污染一直是一个js界争论的焦点，随着html5的盛行，webApp的复杂度成倍增加，对保持原生对象的零污染原则非常重要。
